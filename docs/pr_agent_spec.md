# 広報担当エージェント (PR Agent) 仕様書

## 🤖 概要
広報担当エージェント(PR Agent)は、NPOの広報活動を支援するための自律型AIアシスタントです。
ユーザー（NPO代表者）と対話し、SNS情報を記憶・管理し、定期的な活動サマリの作成や、共感を呼ぶSNS投稿記事のドラフト作成を行います。

## ✨ 主な機能

### 1. SNS情報の記憶・管理
ユーザーが運用しているSNS（Facebook, Instagramなど）のURLや、注目している関連キーワードを記憶します。

### 2. 月次活動サマリの作成
毎月1日の朝9:00に、前月の活動内容や助成金申請状況をまとめたサマリを自動で作成し、Discordに通知します。手動でいつでも作成することも可能です。

### 3. SNS投稿記事のドラフト作成 (Facebook / Instagram)
イベントの写真や企画書の内容を元に、プラットフォームごとの特性に合わせた投稿記事の下書きを作成します。
- **Facebook向け**: ストーリー性を重視した長文、リンク誘導、コミュニティとの対話
- **Instagram向け**: 画像を引き立てるキャッチーな短文、効果的なハッシュタグ選定

### 4. 関連情報の検索
NPOのミッションや関心のあるキーワードに基づいて、Google検索を活用し、最新のトレンドや他団体の事例を収集・報告します。

---

## 🛠️ 使い方とコマンド一覧

Discordチャットでメンション(`@Shadow Director`)して、以下のコマンドまたは自然言語で指示してください。

| 機能 | コマンド例 | 説明 |
| :--- | :--- | :--- |
| **SNS情報の登録** | `FacebookのURLを記憶して: https://facebook.com/npo-sample` | 指定したURLをプロファイルに保存します。 |
| **SNS記事作成** | `Facebook用の投稿記事を作って` <br> (画像またはテキストと共に) | アップロードした画像やテキストを元に、投稿文案を作成します。「インスタ用」と言えばInstagram向けになります。 |
| **月次サマリ** | `今月の活動サマリを作って` | 直近の活動履歴からレポートを作成します。（毎月1日に自動実行もされます） |
| **関連情報検索** | `SDGs教育に関する最新ニュースを探して` | 指定キーワードに関連する情報を検索して報告します。 |

---

## 📅 自動スケジュール機能

システムは以下のスケジュールで自律的に動作します。

- **毎日 9:00**: 日付チェックを実行
- **毎月 1日 9:00**: **月次活動サマリ**を自動生成し、通知します。

---

## 💾 データ管理 (Soul Profile)

PR Agentが扱う情報は、ユーザーごとの `Soul Profile` の一部として統合管理されます。

```json
{
  "user_id": "...",
  "insights": { ... },
  "sns_watch_info": {
    "facebook": "https://facebook.com/...",
    "instagram": "https://instagram.com/..."
  },
  "monthly_summaries": [
    {
      "date": "2025-01-01T09:00:00",
      "summary": "..."
    }
  ]
}
```

## 🔒 セキュリティとプライバシー
- 実際のSNSへの自動投稿機能はありません（下書き作成のみ）。
- 誤投稿を防ぐため、最終的な投稿は必ず人間が確認・実行する必要があります。
- SNSのログイン情報（ID/PASS）は保存しません。公開URLのみを扱います。
